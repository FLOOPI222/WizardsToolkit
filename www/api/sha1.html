DestroySHA1Info

DestroySHA1Info() zeros memory associated with the SHA1Info structure.

The format of the DestroySHA1Info method is:

  SHA1Info *DestroySHA1Info(SHA1Info *sha_info)

A description of each parameter follows:

sha_info

    The cipher sha_info.

sha_info
