GetWizardMagick

GetWizardMagick() produces a unique signature useful for verifying that a structure or file is in fact Wizard related.

The format of the GetWizardMagick method is:

  StringInfo GetWizardMagick(const char *signature,const size_t length)

A description of each parameter follows:

signature

    Derive the magick from this signature.
